# Tестового заданиe

## Описание

Для настройки перед запуском необходимо заполнить необходимые ключи в .env файле

```bash
WB_HEADER_API_KEY=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
SHEETS="" # id для таблиц разделяются пробелами
```
## Запуск проекта

```bash
docker compose up
```
## После запуска
Для обновления данных в таблицах необходимо авторизоваться в Google\
Для авторизации необходимо перейти по ссылке http://localhost:5000/login
